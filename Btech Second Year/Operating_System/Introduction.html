Objectives of This Course

To provide basic knowledge of computer operating system structures and functioning including CPU scheduling, memory management,concurrent processes, deadlocks, security, and integrity.

Outcome of this Course

->Identify the services provided by operating system.

->Understand the internal structure of an operating system and be able to write programs using system calls.

->Understand and solve problems involving process control, mutual exclusion, deadlock and synchronization.
